# AGENTS.md

## Project Setup
This is a React app using Vite, Convex, and WorkOS.

## Development & Testing
- Run `npm install` to install dependencies.
- Run `npx convex dev` requires the CONVEX_DEPLOYMENT env var (provided in secrets).
- To test: Run `npm test`.

## Dependencies
- @convex-dev/react
- @workos-inc/node
